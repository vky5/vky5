## Systems & Infrastructure Projects

### [Blacktree](https://github.com/vky5/Blacktree)

Container deployment platform that builds, schedules, and runs applications directly from GitHub repositories.

**Core work**

* Go-based orchestration runtime
* Worker coordination using gRPC + RabbitMQ
* Concurrent deployment execution
* Health checks and failure-aware scheduling
* Docker image build pipeline
* Automated redeployments

**Focus Areas**
`distributed coordination` `container runtimes` `job scheduling` `deployment orchestration`

---

### [FluxDB](https://github.com/vky5/FluxDB)

Distributed key-value database designed for low-latency replication and state synchronization across nodes.

**Core work**

* Replication protocol design
* Consistency + synchronization strategies
* Concurrent storage engine experimentation
* Failure recovery mechanisms
* Node communication architecture

**Focus Areas**
`distributed databases` `replication` `consensus research` `storage systems`

---

### [FaultLab](https://github.com/vky5/FaultLab)

Deterministic fault injection framework for testing distributed systems under network and node failures.

**Core work**

* Network partition simulation
* Node crash orchestration
* Runtime fault scheduling
* Failure reproducibility tooling
* Distributed system behavior analysis

**Focus Areas**
`chaos engineering` `failure testing` `distributed reliability`
